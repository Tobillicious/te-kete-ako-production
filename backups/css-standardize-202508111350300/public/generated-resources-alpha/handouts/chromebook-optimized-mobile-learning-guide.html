    1. Accessibility Features:
      • Enable ChromeVox (screen reader)
      • Configure high contrast mode
      • Set up dictation tools
    2. Productivity Tools:
      • Organize Google Drive with color coding
      • Set up offline access for key apps
      • Create multiple user profiles for different tasks
    3. Cultural Considerations:
      • Install te reo MƒÅori keyboard
      • Bookmark MƒÅori language resources
