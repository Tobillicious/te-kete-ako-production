    Coding Projects Inspired by Māori Patterns

    Connecting Digital Technology with Traditional Māori Art Forms

    Introduction

    This resource explores how traditional Māori patterns and designs can inspire creative coding projects. By combining digital technology with cultural elements, we can create meaningful connections between past and present, while developing important programming skills.

    Cultural Note: Māori patterns (whakairo) are more than just decorative elements. They carry cultural narratives, represent whakapapa (genealogy), and connect to spiritual beliefs. When using these patterns in coding projects, we must approach them with respect and understanding.

    By completing these activities, students will be able to:

    • Identify key elements of traditional Māori patterns
    • Apply computational thinking to recreate cultural patterns
    • Use programming concepts (loops, functions, variables) to generate patterns
    • Create digital artworks that respectfully incorporate Māori design principles
    • Explain the cultural significance of the patterns they're working with

    Māori Pattern Examples

    Here are some traditional Māori patterns that can inspire coding projects:

    Kōwhaiwhai

    Flowing, curvilinear patterns often seen in meeting houses

    Tāniko

    Geometric weaving patterns with symbolic meanings

    Pūrapura Whetū

    Star cluster patterns representing ancestors

    Coding Activities

    Activity 1: Creating Kōwhaiwhai with Loops

    Use repetition (loops) in programming to create flowing Kōwhaiwhai patterns.

    // Example in JavaScript/p5.js
    function setup() {
      createCanvas(400, 200);
    }

    function draw() {
      background(245, 235, 220); // Natural background
      stroke(0);
      noFill();
      
      // Create repeating koru pattern
      for (let i = 0; i < 8; i++) {
        let x = 50 + i * 40;
        let size = 30 + sin(frameCount * 0.05 + i) * 5;
        drawKoru(x, 100, size);
      }
    }

    function drawKoru(x, y, size) {
      // Draw a simple koru shape
      beginShape();
      vertex(x, y);
      bezierVertex(x + size, y - size/2, x + size/2, y + size, x, y);
      endShape();
    }

    Try It Yourself

    Activity 2: Tāniko Grid Generator

    Create geometric Tāniko patterns using nested loops and conditional statements.

    // Tāniko pattern generator
    function drawTaniko() {
      let cellSize = 20;
      let rows = 10;
      let cols = 10;
      
      for (let r = 0; r < rows; r++) {
        for (let c = 0; c < cols; c++) {
          // Alternate pattern based on position
          if ((r + c) % 2 == 0) {
            fill(0); // Black
          } else {
            fill(200, 0, 0); // Red
          }
          rect(c * cellSize, r * cellSize, cellSize, cellSize);
        }
      }
    }

    Cultural Context & Safety

    Respectful Practice Guidelines:

    • Always acknowledge the Māori cultural origins of the patterns
    • Research and understand the meaning behind patterns before using them
    • Consult with Māori experts or cultural advisors when possible
    • Never use sacred patterns (tapu) without proper guidance
    • Consider creating your own patterns inspired by traditional designs rather than copying exact sacred patterns

    Māori patterns are not merely decorative - they tell stories, represent whakapapa (genealogy), and connect to spiritual beliefs. When used in digital contexts, we must maintain this respect for their cultural significance.

    Assessment Suggestions

    Technical Implementation

    Evaluate how effectively students use programming concepts (loops, functions, conditionals) to create patterns

    Cultural Understanding

    Assess students' ability to explain the cultural significance of the patterns they're working with

    Creativity & Problem-Solving

    Evaluate how students adapt and modify patterns while maintaining cultural integrity

    Documentation

    Review students' explanations of their code and the cultural elements incorporated

    Extension Activities

    For Advanced Learners

    • Create an interactive pattern generator with user-controlled parameters
    • Develop a pattern recognition system that identifies Māori pattern elements
    • Code an animation that tells a pūrākau (Māori narrative) through pattern transitions
    • Build a web application that allows users to create custom Māori-inspired patterns
    • Research and implement more complex patterns like those found in tukutuku panels

    Adaptations for Different Learners

    Supporting Diverse Needs

    For Students Needing More Support:

    • Provide pre-written code templates with fill-in-the-blank sections
    • Use block-based programming (Scratch) before transitioning to text-based code
    • Offer pattern templates that students can modify with simple parameter changes
    • Provide visual step-by-step guides with screenshots

    For Gifted and Talented Students:

    • Challenge them to research and implement lesser-known Māori patterns
    • Encourage development of pattern generation algorithms with mathematical complexity
    • Have them create tutorials or mentor other students
    • Challenge them to connect patterns to specific Māori narratives or whakapapa

    For Students with Physical Disabilities:

    • Ensure all interactive elements are keyboard accessible
    • Provide voice-controlled coding options where possible
    • Offer alternative input methods like touch screens or adaptive devices

    Teacher Implementation Guide

    Step-by-Step Implementation

    1. Preparation (1 session): Introduce Māori patterns and their cultural significance. Show examples and discuss respectful use.
    2. Coding Basics (1-2 sessions): Review or introduce fundamental programming concepts (variables, loops, functions).
    3. Pattern Exploration (2 sessions): Guide students through creating simple patterns using code. Start with basic geometric shapes.
    4. Cultural Patterns (2-3 sessions): Have students implement Māori-inspired patterns using the coding techniques learned.
    5. Project Development (2-3 sessions): Students create their own pattern-based projects with cultural elements.
    6. Sharing & Reflection (1 session): Students present their work and reflect on what they learned about both coding and Māori culture.
