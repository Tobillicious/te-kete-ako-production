Teacher Implementation Notes

Suggested Teaching Approaches:

  • Introduce 5-10 terms per week, building gradually
  • Use the terms consistently in classroom instruction
  • Create word walls with the MƒÅori terms prominently displayed
  • Pair students to quiz each other on terms and definitions
  • Incorporate terms into your regular math problems and discussions

Differentiation Strategies:

  • For beginners: Focus on recognition and pronunciation
  • For intermediate: Practice using terms in simple sentences
  • For advanced: Create original problems using the terms
  • For fluent speakers: Explore deeper cultural connections

Assessment Ideas:

  • Matching exercises (MƒÅori to English terms)
  • Written explanations of concepts using both languages
