    function drawKoru(x, y, size) {
      // Draw a simple koru shape
      beginShape();
      vertex(x, y);
      bezierVertex(x + size, y - size/2, x + size/2, y + size, x, y);
      endShape();
    }

    Cultural Context & Safety

    Respectful Practice Guidelines:

    • Always acknowledge the MÄori cultural origins of the patterns
    • Research and understand the meaning behind patterns before using them
    • Consult with MÄori experts or cultural advisors when possible
    • Never use sacred patterns (tapu) without proper guidance
    • Consider creating your own patterns inspired by traditional designs rather than copying exact sacred patterns

    MÄori patterns are not merely decorative - they tell stories, represent whakapapa (genealogy), and connect to spiritual beliefs. When used in digital contexts, we must maintain this respect for their cultural significance.

    Adaptations for Different Learners

    Supporting Diverse Needs

    For Students Needing More Support:

    • Provide pre-written code templates with fill-in-the-blank sections
    • Use block-based programming (Scratch) before transitioning to text-based code
    • Offer pattern templates that students can modify with simple parameter changes
    • Provide visual step-by-step guides with screenshots

    For Gifted and Talented Students:

    • Challenge them to research and implement lesser-known MÄori patterns
    • Encourage development of pattern generation algorithms with mathematical complexity
    • Have them create tutorials or mentor other students
    • Challenge them to connect patterns to specific MÄori narratives or whakapapa

    For Students with Physical Disabilities:

    • Ensure all interactive elements are keyboard accessible
    • Provide voice-controlled coding options where possible
    • Offer alternative input methods like touch screens or adaptive devices

    Teacher Implementation Guide

    Step-by-Step Implementation

    1. Preparation (1 session): Introduce MÄori patterns and their cultural significance. Show examples and discuss respectful use.
    2. Coding Basics (1-2 sessions): Review or introduce fundamental programming concepts (variables, loops, functions).
    3. Pattern Exploration (2 sessions): Guide students through creating simple patterns using code. Start with basic geometric shapes.
    4. Cultural Patterns (2-3 sessions): Have students implement MÄori-inspired patterns using the coding techniques learned.
    5. Project Development (2-3 sessions): Students create their own pattern-based projects with cultural elements.
    6. Sharing & Reflection (1 session): Students present their work and reflect on what they learned about both coding and MÄori culture.
