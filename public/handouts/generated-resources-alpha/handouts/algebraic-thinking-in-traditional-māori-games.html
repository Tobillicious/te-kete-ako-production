Activity 1: Mu Torere Strategy Analysis

Game Overview

Mu torere is a traditional MÄori board game played on an 8-pointed star (pÅ«tahi) with 8 playing pieces (4 per player).

Algebraic Challenge

After playing several rounds, identify:

  1. How many possible first moves exist?
  2. If each player has n pieces, how would you calculate possible moves?
  3. Create an equation to predict the minimum number of moves to win

Teacher Implementation

  1. Introduce the game with a demonstration (use drawn board or digital version)
  2. Have students play in pairs (15 minutes)
  3. Facilitate discussion about strategies and patterns
  4. Introduce algebraic representation of moves
  5. Challenge students to create their own equations

Activity 2: Poi RÄkau Patterns

Game Overview

Poi rÄkau involves rhythmic passing of sticks in patterns. We'll analyze the mathematics behind these patterns.

Pattern Analysis

With a partner, try these passing sequences:

Sequence Algebraic Representation
Pass every 2 beats t = 2n
Pass on prime-numbered beats t = p (where p is prime)

Challenge: Create your own sequence and represent it algebraically

Teacher Implementation

  1. Demonstrate basic poi rÄkau rhythms (use rulers or rolled paper if sticks unavailable)
  2. Start with simple patterns, gradually increasing complexity
  3. Connect physical patterns to numerical representations
  4. Have students create and share their own algebraic patterns
