Te Mahi Tuatahi | Activity 1: Mu T≈çrere Strategy Analysis

Game Setup

Mu T≈çrere is played on an 8-pointed star with a center point. Each player has 4 pieces placed on opposite sides of the star.

Strategic Thinking Challenge

If Player A has a winning strategy that involves moving pieces in a specific sequence, and Player B responds with counter-moves:

Let x = number of moves until Player A can force a win

Let y = number of defensive moves Player B can make

If x = 2y - 3, and Player B has 5 defensive moves available, how many moves until Player A wins?

Algebraic Representation

Create algebraic expressions to represent:

  • The total number of possible moves from any position
  • The relationship between pieces on the board and available moves
  • Predicting the outcome based on current piece positions

Te Mahi Tuarua | Activity 2: Whai Pattern Sequences

Pattern Analysis

Whai (string games) create geometric patterns through specific sequences of moves.

Pattern Name Number of Moves Mathematical Relationship
Whai (Simple Diamond) 4 M = 2n (where n = number of diamonds)
Matariki (Seven Sisters) 7 M = n + (n-1) (where n = number of stars)
Te Ika (The Fish) 10 M = 3n - 2 (where n = number of fish elements)

Pattern Prediction

If a complex Whai pattern requires 22 moves to complete, and follows the formula M = 4n - 2, how many elements (n) does the pattern have?
