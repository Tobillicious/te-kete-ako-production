📖 Usage Guidelines

🖨️ Printing Instructions

  • All worksheets are optimized for A4 printing
  • Teacher answer keys appear on screen but won't print
  • Use "Print" button or Ctrl/Cmd+P to print
  • Adjust margins if needed in print dialog

🏛️ Cultural Protocols

  • Review cultural content with advisors before use
  • Practice te reo Māori pronunciation properly
  • Ensure respectful treatment of traditional knowledge
  • Consult with local iwi when appropriate

♿ Accessibility

  • Digital versions work with screen readers
  • Simplified language versions available for EAL
  • Contact for additional accessibility formats

🎯 Differentiation Support
