    Hands-on Activity: Hybrid Navigation Model

    Task: Design a navigation system that combines traditional MÄori knowledge with modern GPS technology.

    Steps:

    1. Identify 3 traditional navigation methods that could complement GPS
    2. Explain how each method would provide redundancy if GPS failed
    3. Create a diagram of your hybrid system
    4. List advantages of your integrated approach

    Materials:

    • Paper/pencils or digital drawing tools
    • Star charts (provided)
    • Ocean current maps (provided)
    • GPS demonstration (phone app)
