Te Reo Māori Project: Create Your Own Learning Resource

"Mā te ako ka mōhio, mā te mōhio ka mārama"

"Through learning comes knowledge, through knowledge comes understanding"

Project Overview

Task: Create an educational resource to help others learn te reo Māori

Duration: 3-4 weeks

Format: Your choice (see options below)

Project Options

1. Video Series

Create 3-5 short videos teaching specific te reo topics (greetings, numbers, etc.)

2. Mobile App/Game

Design a simple te reo learning app or game (can be prototype)

3. Children's Book

Write and illustrate a bilingual children's book

4. Podcast Series

Create 3+ podcast episodes about te reo and culture

5. Interactive Website

Build a website with lessons, quizzes, and resources

6. Board Game

Design a physical board game for practicing te reo

Project Requirements

All projects must include:

  • ✅ Accurate te reo Māori (proofread with teacher/fluent speaker)
  • ✅ Cultural appropriateness (respectful representation)
  • ✅ Clear learning goals (what will users learn?)
  • ✅ Engaging format (creative, interactive, user-friendly)
  • ✅ Authentic assessment (how do users know they've learned?)
  • ✅ Accessibility (suitable for target audience)
