YEAR 7 MATHEMATICS

Introduction to Patterns & Algebra

📐 Mathematics • ⏱️ 60 minutes • 👥 Year 7 • 🌿 Cultural: Tukutuku patterns

🏫 School Values Connection

🎯 Excellence
Pattern recognition builds mathematical thinking
🌿 Respect
Honoring tukutuku weaving traditions
🤝 Community
Collaborative pattern discovery

🎯 WALT (We Are Learning To)

  • Recognize and describe patterns in number sequences
  • Identify rules that generate patterns
  • Use algebraic notation to describe patterns
  • Connect mathematical patterns to cultural designs

✅ Success Criteria

  • I can extend a number pattern by at least 3 terms
  • I can describe the rule in words
  • I can use "n" to write a pattern rule
  • I can create my own pattern inspired by tukutuku

📚 Lesson Flow

1️⃣ Introduction (10 minutes)

Hook: Display images of tukutuku panels from various marae. Ask students: "What do you notice about these patterns? How are they the same? How are they different?"

Cultural Context:

Tukutuku panels are woven wall panels found in wharenui (meeting houses). Each pattern has a name and meaning. The patterns follow mathematical rules - perfect for introducing algebra!

Key Question: "Can you predict what comes next in a pattern?"

2️⃣ Activity 1: Number Patterns (15 minutes)

Pairs work: Students identify patterns and describe the rules.

Pattern Examples:
  • Pattern A: 2, 5, 8, 11, 14, ... (add 3 each time)
  • Pattern B: 1, 4, 9, 16, 25, ... (square numbers)
  • Pattern C: 3, 6, 12, 24, 48, ... (double each time)
  • Pattern D: 100, 95, 90, 85, 80, ... (subtract 5 each time)

Teacher circulates: Ask "What's the rule?" and "How did you figure it out?"

3️⃣ Activity 2: Introducing Algebra (15 minutes)

Teacher models: Show how to use "n" to represent position in a pattern.

Example:

Pattern: 5, 10, 15, 20, 25, ...

Rule in words: "Multiply the position by 5"

Rule with algebra: n × 5 or 5n

Position (n) Term (5n)
1 5
2 10
3 15
10 50

Students practice: Convert 3 patterns from Activity 1 into algebraic rules.

4️⃣ Activity 3: Create Your Own Pattern (15 minutes)

Creative task: Students design their own tukutuku-inspired pattern using mathematical rules.

Instructions:
  1. Choose a starting number
  2. Choose a rule (add, subtract, multiply, or a combination)
  3. Generate at least 6 terms
  4. Write the rule in words AND using "n"
  5. Draw a simple visual pattern that matches your number pattern

Extension: Can you create a pattern where the rule involves adding AND multiplying? (e.g., 2n + 1)

5️⃣ Conclusion & Reflection (5 minutes)

Class discussion: Gallery walk of student-created patterns. Students explain their rules to a partner.

Exit ticket: "What is one pattern you see in everyday life? Describe its rule."

📊 Assessment

Formative Assessment:
  • Observe pair work discussions - Are students identifying rules correctly?
  • Check algebraic notation - Can students use "n" appropriately?
  • Review created patterns - Do they demonstrate understanding?
Success Indicators:
  • ✅ Students extend patterns accurately
  • ✅ Students articulate rules verbally
  • ✅ Students use algebraic notation (5n, 3n + 2, etc.)
  • ✅ Students connect math to cultural patterns

📦 Resources Needed

  • Images of tukutuku panels (print or digital)
  • Whiteboard/smartboard for modeling
  • Paper and colored pencils for pattern creation
  • Optional: Pattern blocks or manipulatives

💡 Teacher Notes

  • Differentiation: Struggling students can work with simpler patterns (add/subtract only). Advanced students can explore quadratic patterns (n²).
  • Cultural note: If possible, invite a local weaver to speak about tukutuku patterns and their meanings.
  • Extension: Next lesson can explore graphing these patterns on coordinate planes.
  • Common misconception: Students might confuse the pattern rule with the term number. Use tables to clarify.
