📚 Lesson Flow

1️⃣ Introduction (10 minutes)

Hook: "How many of you have played quiz games or apps? Kahoot? Quizlet? Today YOU become the creator! We'll build a te reo quiz game using code."

Why Code?

Coding = giving instructions to a computer. Just like you follow steps in a recipe, computers follow code. Today we use Scratch (block-based coding - easier than typing text code). This is computational thinking in action!

2️⃣ Activity 1: Scratch Basics (15 minutes)

Teacher demo + students follow along: Open Scratch, explore interface.

Key Concepts:
  • Variables: Store information (e.g., score, player name)
  • Ask & Answer: Blocks that ask questions and get user input
  • If/Else: Make decisions (if answer correct, add point; else, say wrong)
  • Broadcast: Send messages between sprites (like walkie-talkies!)

Practice: Make sprite ask "He aha tō ingoa?" (What's your name?) and respond "Kia ora, [name]!"

3️⃣ Activity 2: Build the Quiz! (25 minutes)

Guided coding: Build quiz step-by-step together.

Quiz Structure (Pseudocode): 
1. Create variable "score" (set to 0)
2. Say "Welcome to Te Reo Quiz!"
3. Ask "What is 'hello' in te reo?"
4. If answer = "kia ora" then:
   - Say "Tika! (Correct!)"
   - Change score by 1
5. Else:
   - Say "Hē! (Wrong!) It's kia ora"
6. Ask "What is 'goodbye'?"
7. If answer = "ka kite" or "haere ra" then:
   - Say "Tika!"
   - Change score by 1
8. Else:
   - Say "Hē! It's ka kite or haere ra"
9. Ask "What is 'thank you'?"
10. If answer = "ngā mihi" or "kia ora" then:
    - Say "Tika!"
    - Change score by 1
11. Else:
    - Say "Hē! It's ngā mihi"
12. Say "Final score: [score] out of 3!"

Extension: Add sound effects, background, more questions, timer!

4️⃣ Testing & Debugging (5 minutes)

Test your quiz: Play it! Does it work? Find bugs!

Common Bugs:
  • Spelling mistakes in answer checks (case sensitive!)
  • Score not updating (check "change score by 1" block)
  • Questions not appearing in order (check order of blocks)

5️⃣ Showcase & Reflect (5 minutes)

Gallery walk: Play 2-3 classmates' quizzes. What's cool? What could improve?

Reflection: "How could coding help revitalize te reo Māori?"

🌿 Cultural Depth: Technology for Language Revitalization

Te reo Māori nearly died out (by 1980s, <2% of Māori children fluent). Technology now helps SAVE it:

  • Apps: Kupu (dictionary), Drop (game), Te Aka (online dictionary)
  • Platforms: Te Reo Singalong (YouTube), Māori TV streaming
  • AI: Google Translate now supports te reo, Siri speaks Māori
  • Games: Like what YOU just built! Fun = engagement = learning
  • Your mahi: By coding in/for te reo, YOU support language revitalization! 🌟
