GET /api/resources

Retrieve educational resources with optional filtering.

Query Parameters:

Parameter Type Description
subject string Filter by subject (Mathematics, Science, English, etc.)
year_level string Filter by year level (7, 8, 9, 10, 11, 12, 13)
quality_min number Minimum quality score (0-100)
limit number Number of results (default: 20, max: 100)

Example Request:

GET /api/resources?subject=Mathematics&year_level=9&quality_min=90&limit=10

Example Response:

{ "success": true, "data": [ { "id": "123e4567-e89b-12d3-a456-426614174000", "title": "Algebra Pattern Recognition", "description": "Complete algebra unit with cultural patterns", "subject": "Mathematics", "year_level": "9", "quality_score": 95, "path": "/units/y9-maths-algebra/index.html", "cultural_elements": { "has_whakataukƒ´": true, "has_te_reo": true, "integration_score": 92 } } ], "count": 10, "total": 150 }

GET /api/resources/:id

Get detailed information about a specific resource.

Example Request:

GET /api/resources/123e4567-e89b-12d3-a456-426614174000

GET /api/pathways

Retrieve prerequisite learning pathways for a resource.

Example Request:

GET /api/pathways?resource_id=123e4567-e89b-12d3-a456-426614174000

Example Response:

{ "success": true, "pathway": [ { "step": 1, "title": "Introduction to Patterns", "id": "abc123...", "confidence": 0.95 }, { "step": 2, "title": "Basic Algebra Concepts", "id": "def456...", "confidence": 0.92 } ] }

POST /api/progress

Record student progress on a resource. (School/Enterprise plans only)

Request Body:

{ "student_id": "student-uuid", "resource_id": "resource-uuid", "progress_percentage": 75, "score": 18, "max_score": 20, "time_spent_minutes": 45 }
