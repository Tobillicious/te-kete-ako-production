📖 Unit Overview

Learning Journey

This unit introduces algebraic thinking through culturally relevant contexts. Students explore patterns in kowhaiwhai and tukutuku designs, learn to use variables through traditional Māori games, and solve equations using culturally responsive pedagogies.

🎯 Key Objectives:

  • Recognize and extend patterns using algebraic thinking
  • Understand variables as unknown quantities
  • Write and solve simple algebraic equations
  • Connect algebra to traditional Māori patterns and games

🌿 Cultural Context
