üîÑ The Settlement Process (7 Steps)

  1. File Claim: Iwi files claim with Waitangi Tribunal
  2. Investigation: Tribunal researches historical evidence
  3. Findings: Tribunal determines if Treaty was breached
  4. Negotiations: Crown and iwi negotiate redress package
  5. Settlement: Financial compensation + land return + apology
  6. Legislation: Parliament passes settlement act
  7. Implementation: Assets transferred, iwi invests for future
