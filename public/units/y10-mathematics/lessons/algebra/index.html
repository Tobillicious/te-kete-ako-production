🎯 Learning Objectives

📚 Knowledge Objectives

  • Solve quadratic equations using multiple methods
  • Understand function relationships and graphical representation
  • Model real-world scenarios with systems of equations
  • Apply polynomial operations to complex problems
  • Use exponential functions to model growth and decay

🌿 Cultural Integration

  • Connect mathematical patterns to traditional Māori art forms
  • Apply algebraic thinking to environmental kaitiakitanga
  • Use mathematical modeling for community resource allocation
  • Explore exponential growth in cultural knowledge transmission
  • Develop financial literacy with Māori economic principles

📊 Assessment Framework

✅ Formative Assessment

  • Daily exit tickets with cultural reflection
  • Peer feedback on problem-solving approaches
  • Progress checks on algebraic manipulation skills
  • Self-assessment of cultural connections made

🎯 Summative Assessment

  • Unit test covering all algebraic concepts
  • Mathematical modeling project with cultural context
  • Portfolio of solved problems with explanations
  • Reflection on cultural learning connections
