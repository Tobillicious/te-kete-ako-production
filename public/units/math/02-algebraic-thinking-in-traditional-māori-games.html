Traditional Māori Games with Algebraic Thinking

Mu Tōrere

Strategic board game from Te Arawa

Mu Tōrere is a strategy game played on a star-shaped board (papa tākaro) with eight points (kāwai) and one center (pū). Players move pieces according to specific rules, creating patterns that demonstrate algebraic thinking.

Algebraic Concepts:

  • Pattern Recognition: Identifying winning move sequences
  • Variables: Representing game positions with symbols
  • Functions: Understanding how moves change game state
  • Strategic Thinking: Planning multiple moves ahead

Activity: Algebraic Analysis

1
Represent Positions

Create algebraic expressions to represent different game positions

2
Pattern Analysis

Identify patterns in winning strategies using algebraic notation

3
Strategy Development

Develop algebraic rules for optimal gameplay

Whai (String Games)

Traditional string figures and patterns

Whai are string games that create intricate patterns through sequential movements. These patterns demonstrate geometric and algebraic concepts through their creation and transformation.

Algebraic Concepts:

  • Sequences: Understanding ordered steps in pattern creation
  • Transformations: How patterns change with different movements
  • Symmetry: Identifying symmetrical properties
  • Variables: Representing different string positions

Activity: Pattern Algebra

1
Sequence Notation

Create algebraic notation for string pattern sequences

2
Pattern Variables

Use variables to represent different string positions

3
Transformation Rules

Develop algebraic rules for pattern transformations

Kī-o-Rahi

Team sport incorporating mathematical thinking

Kī-o-Rahi is a traditional Māori team sport played on a circular field with specific zones. The game involves strategic positioning and movement that demonstrates algebraic concepts.

Algebraic Concepts:

  • Coordinate Systems: Understanding positioning on the field
  • Variables: Representing player positions and movements
  • Functions: Describing movement patterns
  • Strategy: Optimizing team formations

Activity: Strategic Algebra

1
Position Variables

Use variables to represent player positions

2
Movement Functions

Create functions describing optimal movement patterns

3
Strategy Optimization

Use algebraic thinking to optimize team strategies

Mātauranga Māori Integration

Whakapapa of Knowledge

These games carry whakapapa - genealogy that connects mathematical thinking to ancestral knowledge. Each game represents generations of mathematical thinking embedded in cultural practice.

Māori Ways of Knowing

The algebraic thinking in these games demonstrates mātauranga Māori - Māori knowledge systems that integrate mathematical concepts with cultural values and practices.

Contemporary Relevance

By understanding the algebraic thinking in traditional games, students connect mathematical concepts to their cultural identity and contemporary applications.
