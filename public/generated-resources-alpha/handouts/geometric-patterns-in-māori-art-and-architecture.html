Key Concepts

1. K≈çwhaiwhai (Rafter Patterns)

These repeating patterns use transformations:

  • Reflection symmetry (like the koru spiral)
  • Translation (repeating units)

2. Tukutuku (Lattice Panels)

Mathematical features:

  • Grid-based designs (Cartesian coordinates)
  • Tessellation (repeating shapes with no gaps)

Activity: Design Your Own K≈çwhaiwhai

Materials: Grid paper, rulers, compasses, digital design tools (optional)

  1. Choose a base shape (e.g., koru, triangle)
  2. Apply one transformation (reflection/rotation/translation)
  3. Repeat to create a repeating pattern
  4. Explain the mathematical rules used
