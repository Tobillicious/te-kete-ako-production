Coding Projects Inspired by Māori Patterns

Cultural Context

Māori patterns (whakairo) carry deep cultural significance, representing stories, ancestors, and connections to the natural world. When working with these patterns in digital contexts, we must approach them with respect and understanding.

Key concepts: Koru (unfurling fern frond - new beginnings), Mangōpare (hammerhead shark - strength), Puhoro (speed and swift movement), Unaunahi (fish scales - abundance).

Learning Objectives

  • Create digital representations of Māori patterns using programming concepts
  • Understand the cultural significance of traditional Māori designs
  • Apply sequencing, iteration, and functions in programming
  • Develop debugging and problem-solving skills
  • Create culturally respectful digital artifacts

Project 1: Digital Koru Generator

Using Scratch to Create a Koru Animation

The koru represents new life, growth, and harmony. We'll create an animation showing multiple koru growing.

Steps:

  1. Open Scratch (scratch.mit.edu) and create a new project
  2. Delete the cat sprite and create a new sprite
  3. Use the pen extension to draw: when green flag clicked clear pen down set pen color to [#2ecc71] set pen size to (3) repeat (10) move (10) steps turn (15) degrees move (10) steps turn (-15) degrees end
  4. Experiment with changing the numbers to create different koru shapes
  5. Add multiple sprites with different starting positions and colors

Try This:

Modify the code to make the koru grow larger over time by increasing the move steps gradually.

Project 2: Python Turtle Māori Patterns

Creating Traditional Patterns with Code

We'll use Python's Turtle module to create geometric patterns inspired by traditional tukutuku panels.

Basic Diamond Pattern Code:

import turtle t = turtle.Turtle() t.speed(0) t.color("red") def diamond(): for _ in range(2): t.forward(50) t.right(60) t.forward(50) t.right(120) for _ in range(12): diamond() t.right(30) turtle.done()

Challenge:

Modify the code to create a pattern that alternates between two colors (like traditional black and red tukutuku).

Extension Activities

For Advanced Learners:

  • Create an interactive web page using p5.js that allows users to generate custom Māori patterns
  • Develop a Python program that takes user input to determine pattern complexity and color schemes
  • Research and implement a more complex pattern like the Mangōpare (hammerhead shark) design
  • Create a digital story that explains the cultural significance of your patterns through animation

Adaptations for Different Learners

For Students Needing More Support:

  • Provide pre-written code with blanks to fill in
  • Use block-based programming (Scratch) before text-based
  • Offer printed step-by-step visual guides
  • Pair students with peer mentors

For Visually Impaired Students:

  • Use screen reader compatible environments
  • Provide tactile versions of patterns
  • Focus on sound-based output (e.g., different tones for different shapes)

Teacher Implementation Guide

Before Teaching:

  • Familiarize yourself with basic Māori pattern meanings (see Resources section)
  • Test all code examples to ensure they work in your school environment
  • Prepare exemplars at different levels for students to reference

Lesson Sequence:

  1. Introduction (30 min): Show examples of Māori patterns and discuss their meanings
  2. Scratch Tutorial (45 min): Guide students through creating basic koru
  3. Python Introduction (45 min): Demonstrate Turtle graphics with simple shapes
  4. Independent Work (2-3 lessons): Students develop their own pattern designs
  5. Sharing (30 min): Students present work and explain cultural connections
