---## **Teacher Implementation Guide**### **1. Preparation**- **Resources Needed:** - Images of Māori artworks (ensure permissions if using sacred pieces) - Access to Māori artist profiles (e.g., from Te Papa’s online collection)- **Pre-lesson:** - Review key Māori art terms (e.g., whakairo, kōwhaiwhai) - Consult with local iwi or school cultural advisors if unsure### **2. Lesson Flow**1. **Introduction (10 min):** Discuss why cultural context matters in art analysis.2. **Guided Analysis (20 min):** Use the handout to analyse an artwork together.3. **Independent Task (15 min):** Students analyse a different artwork in pairs.4. **Sharing (10 min):** Groups present findings, focusing on Māori perspectives.### **3. Assessment Options**- **Formative:** Peer feedback on analysis depth- **Summative:** Written report comparing two artworks### **4. Cultural Safety Reminders**- Avoid generalising Māori art—acknowledge regional differences.- Use te reo Māori correctly (check pronunciation).---This resource is **ready for immediate classroom use**, aligns with the **NZC Arts (Levels 4–5)**, and supports **culturally sustaining pedagogy**.
