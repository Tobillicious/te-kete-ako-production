--- ## **Teacher Implementation Guide** ### **Step-by-Step Instructions** 1. **Preparation:** - Preview media examples for appropriateness. - Ensure digital access (Chromebook-friendly links). 2. **Lesson Flow:** - **Introduction (10 mins):** Discuss media influence. - **Case Study (30 mins):** Analyze clips in small groups. - **Discussion (20 mins):** Share findings as a class. - **Assessment (20+ mins):** Choose a task (written, video, or creative rewrite). 3. **Closing:** - Reflect: "What’s one thing you’ll notice differently in media now?" --- ## **Assessment Rubric** | **Criteria** | **Achieved (✓)** | **Developing (△)** | **Not Yet (✗)** | |----------------------------|------------------|--------------------|-----------------| | Identifies stereotypes | | | | | Analyzes creator perspective | | | | | Proposes better representation | | | | | Clear, respectful communication | | | | --- ## **Cultural & Accessibility Notes** - **Māori Terms:** Use whānau (family), mana (prestige), etc., with explanations. - **Chromebook Tips:** Pre-load videos to avoid buffering; provide transcripts. - **Sensitive Content:** Avoid over-emphasizing negative stereotypes; balance with positive examples. --- **Ready for classroom use!** This resource aligns with NZC English (Levels 4-6) and Media Studies standards while fostering critical thinking and cultural awareness.
