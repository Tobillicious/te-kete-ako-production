Interactive Activity: RongoƒÅ MƒÅori (Traditional Medicine) Investigation

Scenario: MƒÅori traditionally used kawakawa leaves for medicinal purposes. Let's investigate their antibacterial properties using both scientific approaches.

Western Scientific Approach:

  1. Observe that kawakawa seems to prevent infection when applied to wounds
  2. Question: Does kawakawa extract inhibit bacterial growth?
  3. Hypothesis: Kawakawa extract will prevent bacterial growth in petri dishes
  4. Experiment: Create agar plates with different concentrations of kawakawa extract
  5. Measure zones of inhibition after incubation

MƒÅori Knowledge Approach:

  1. Titiro: Observe how ancestors used kawakawa for generations
  2. PƒÅtai: In what situations was it most effective?
  3. Whakapae: The plant is most effective when prepared in specific ways
  4. WhakamƒÅtautau: Test different preparation methods (fresh vs dried, different seasons)
  5. TƒÅtari: Consider environmental factors like time of harvest

Teacher Implementation Guide

Before the Lesson:

  1. Consult with MƒÅori community members about appropriate content
  2. Prepare kawakawa leaves (ensure proper harvesting protocols if collecting fresh)
  3. Set up lab equipment for antibacterial testing
  4. Create visual aids comparing the two methodologies

Lesson Sequence (2-3 sessions):

  1. Introduction (30 min):
    • Discuss what science is - broaden understanding beyond Western paradigm
    • Introduce MƒÅori scientific traditions through examples (navigation, medicine, agriculture)
    • Compare the two methodologies using the table provided
  2. Practical Investigation (60-90 min):
    • Divide class into groups - some use Western method, some use MƒÅori approach
    • Guide through investigation protocols for both methods
    • Emphasize respectful handling of MƒÅori knowledge
  3. Analysis and Reflection (60 min):
    • Groups share findings
    • Discuss strengths of each approach
    • Journal reflection: How can both knowledge systems work together?
